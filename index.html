All Routes available

  • GET /books : get all books in the table
  • GET /books/:id : get a book given an id
  • DELETE /books/:id : delete a book given an id
  • POST /books : create a new book in the table, must have name and author in the request body
  • PUT /books/:id : update a book given the id, must have name, author, and haveRead status (true or false) in the request body
